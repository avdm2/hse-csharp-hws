Условие 9: задать формат выделенного текста. В приложении данный пункт
подразумевается, как изменение стиля (жирный, курсив, подчеркнутый, 
зачеркнутый, по умолчанию) написанного текста.

Стиль "По умолчанию" сбрасывает все примененные на текст стили. 

Измененный стиль шрифта может быть сброшен на значение по 
умолчанию через соответствующий пункт подменю. При повторном нажатии
на примененный стиль ничего не произойдет.

К счастью или к сожалению, изображение вставить не получится.

Размер шрифта задается через пункт "Выбор шрифта и размера" в меню формата.
Выборы размера текста и шрифта реализованы через стандартный fontDialog.

Под изменением выбора цветовой схемы приложения подразумевается смена
цвета шрифта и заднего фона. При смене цвета заднего фона цвет шрифта
автоматически меняется на более читабельный. Впоследствии его тоже можно 
сменить на один из предложенных.

Наиболее корректно программа работает с файлами следующих текстовых типов:
.rtf, .txt.

Функционал: 
1. Открывать, редактировать и сохранять простые текстовые файлы (без сохранения 
форматирования).
2. Открывать, редактировать и сохранять текстовые файлы формата *.rtf (с 
сохранением форматирования). 
3. При наличии несохраненных изменений в открытых файлах при закрытии 
приложения  предоставить пользователю возможность сохранить файл, не сохранять 
или отменить закрытие приложения.
4. Реализовать меню приложения со следующими подменю: Файл, Правка, 
Формат(для  выбора  основного  шрифта  текста), Настройки. Список может быть 
дополнен на ваше усмотрение.
5. Реализовать возможность выбора цветовой схемы приложения в настройках.
6. Предоставить возможность пользователю форматировать выделенный фрагмент 
изменяя стиль текста по следующим параметрам: курсив, жирный, подчеркнутый,
зачеркнутый. Возможно использование нескольких шрифтов сразу.
7. Создать контекстное меню, активируемое правой кнопкой мыши, со следующим 
функционалом: выбрать весь текст, вырезать выделенный фрагмент, копировать 
выделенный фрагмент, вставить сохраненный в буфере обмена фрагмент, задать 
формат выделенного фрагмента текста.
